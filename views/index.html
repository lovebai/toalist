API 文档

上传文件 API 接口说明：

POST /api/upload
Content-Type: multipart/form-data

参数：
- file: 文件（支持多文件上传）
- sign: 时间戳（随机字符也可）

响应：
{
    "code": 200,
    "message": "上传成功",
    "files": [
        {
            "name": "文件名",
            "url": "文件访问地址"
        }
    ]
}

状态码：
- 200: 全部文件上传成功
- 207: 部分文件上传成功
- 400: 请求参数错误
- 413: 文件大小超出限制
- 415: 不支持的文件类型
- 500: 服务器内部错误

关于 ToAlist

ToAlist 是一个简单易用的文件上传服务，支持图片预览和多种格式的链接生成。

主要功能：

  • 支持拖拽上传和点击上传
  • 支持图片文件预览
  • 自动生成 Markdown、HTML、BBS 等多种格式的链接
  • 支持暗色模式
  • 支持多文件同时上传

技术栈：

  • 前端：Vue 3 + Element Plus
  • 后端：Go
